# UI Naming Rules

This project uses a simple page-first naming scheme.

## Pages

- `PageHomeView`
- `PageCharacterView`
- `PageBattleView`
- `PageStoryView`
- `PageActivityView`
- `PageSettingsView`

## Scene names

- `Scene_Boot`
- `Scene_Login`
- `Scene_Home`
- `Scene_Battle`

## Recommended UI folders

- `Assets/Scripts/UI`
- `Assets/Scripts/UI/Pages`
- `Assets/Scripts/UI/Widgets`
- `Assets/Scripts/UI/Transitions`
- `Assets/Art/UI`
- `Assets/Art/Backgrounds`
- `Assets/Art/Portraits`

## Extending later

When a new page is added, keep the same pattern:

- `PageXxxView`
- `PageXxxController`
- `PageXxxModel`
