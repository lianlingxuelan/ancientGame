# Unity UI Prototype

This folder is the first-pass UI engineering skeleton for the project.

Core UI direction:

- Warm, soft, supportive main庭院
- Flexible activity themes for cold-season events
- Game-first navigation with low visual pressure
- Easy-to-extend page naming and folder conventions

Recommended top-level pages:

- Home庭院
- Character角色
- Battle战斗
- Story剧情
- Activity活动
- Settings设置

Recommended page naming rule:

- `Page_<Name>View`
- `Page_<Name>Controller`
- `Page_<Name>Model`

Example:

- `PageHomeView`
- `PageCharacterView`
- `PageBattleView`

Recommended scene naming:

- `Scene_Boot`
- `Scene_Login`
- `Scene_Home`
- `Scene_Battle`
