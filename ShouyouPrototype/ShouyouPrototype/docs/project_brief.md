# 手游项目第一版说明文档

## 1. 项目定位

这是一个以历史人物为核心灵感、偏国风审美、以角色养成和回合 PVE 为主的手游原型项目。

当前方向已经从“纯剧情手游”调整为：

- 角色养成主导
- 回合制 PVE 推进
- 剧情作为驱动和补充
- 情绪价值、角色陪伴感优先
- 梦境圆梦感作为叙事底色

## 2. 当前核心判断

### 2.1 玩法主轴

主轴建议定为：

`角色养成 + 半自动回合 PVE + 剧情解锁`

这样更符合现阶段市场习惯，也更适合你一个人先做第一章 Demo。

### 2.2 剧情定位

剧情不作为主要负担，而是拆成多个层次：

- 主线：短、清晰、推动世界观
- 传记：补充单角色背景
- 羁绊：强化陪伴感
- 支线：扩充人物关系和事件
- CG：关键情绪节点奖励

### 2.3 世界观处理

推荐走“历史灵感 + 轻架空梦境世界”的路线。

原因：

- 能保留李清照、唐婉、张仲景、辛弃疾等人物气质
- 方便后续扩展更多古人
- 能兼顾历史厚度和游戏可玩性

“梦境圆梦”设定适合承接人物遗憾与情绪宣泄。

## 3. UI / UX 审美方向

### 3.1 总体风格

UI 方向建议为：

- 宋代国风
- 柔雾感
- 暖色常驻界面
- 冷色节日/活动界面
- 大留白
- 低压迫感
- 角色陪伴感优先

### 3.2 视觉关键词

- 庭院
- 回廊
- 月门
- 花枝
- 灯笼
- 纱幔
- 书案
- 玉台
- 轻梦境

### 3.3 主界面建议

主界面不应该信息过载，建议保留：

- 顶部资源栏
- 中央角色展示位
- 底部五入口导航
- 右上角设置/邮件/活动提示

推荐底部导航：

- 庭院
- 角色
- 战斗
- 剧情
- 活动

设置不放在主导航里，放右上角图标。

## 4. 场景方向

### 4.1 常驻暖色场景

适合主界面、日常页、养成页、角色陪伴页。

- 主界面庭院
- 角色书房
- 养成台 / 词意台
- 抽卡祈愿台
- 普通战斗场景
- 章节过场空镜

### 4.2 活动冷色场景

适合冬日活动、节气限定、情绪高潮、Boss 战。

- 雪夜梦境
- 雨夜战斗场景
- 冬日活动庭院
- Boss 专属场景

- `bg_story_cg`
- `bg_seasonal`
- `cg_story`
- `cg_romance`
- `ui_material`
- `character_card`
- `avatar_chibi`
- `reference_sheet`

### 6.2 命名规则

统一采用：

- `类别_序号三位数.jpg`

例如：

- `bg_scene_001.jpg`
- `cg_romance_003.jpg`
- `avatar_chibi_012.jpg`

以后补图只要继续往后编号即可。
