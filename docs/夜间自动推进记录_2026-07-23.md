## 明早你需要在 Unity 里做的检查

1. 打开 Unity。
2. 点击顶部菜单：

   `Shouyou > UI > Clean And Rebuild Prototype`

3. 等 Unity 导入新脚本和资源。
4. 运行游戏。
5. 检查：
   - 庭院能进入主线。
   - 主线页左侧有 4 个栏目。
   - 梦域养成活动按钮能弹出梦域说明。
   - 关卡卡片能点击。
   - 未解锁关卡点击后，开始阅读 / 进入战斗会提示未解锁。
   - `Assets/_Project/Resources/UIThemes` 下是否生成 `RealityTheme` 和 `DreamTheme`。
   - 点击右上角“设置”按钮，观察主题颜色和背景是否发生变化。

## 如果 Unity 报错，优先看这些点

1. 新脚本是否已经被 Unity 导入。
2. `UIThemeConfig` 是否能编译。
3. `HomeUILayoutBuilder.cs` 是否能找到 `Shouyou.UI.Theme` 命名空间。
4. 如果主题资源没有生成，重新点一次一键生成 UI。
